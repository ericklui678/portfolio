A web application that allows users to search for the latest rocket league player stats from PC, Playstation, and Xbox platforms. When registered or logged in, users can also follow and track their own list of players on a single screen for stat comparison.

iOS game that utilizes CoreMotion where the user tilts the phone left and right to move the spatula and flicks the phone upward to "flip" as many burger patties within the time limit using the phone's built-in gyroscope.

  • Technologies Used: Swift | CoreMotion | AVFoundation

  • Source Code: GitHub
